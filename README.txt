Para correr la prueba de forma local: 

No es necesario instalar librerias, pero si es necesario tener conexion a internet, pues las librerias usadas se cargan via CDN.

Las Librerias usadas son: 

Tailwind.css --- Para Estilos.
<script src="https://cdn.tailwindcss.com"></script>

Font-awesome --- Para íconos. 
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.3.0/css/all.min.css" integrity="sha512-SzlrxWUlpfuzQ+pcUCosxcglQRNAq/DZjVsC0lE40xsADsfeQoEypE+enwcOiGjk/bSuGGKHEyjSoQ1zVisanQ==" crossorigin="anonymous" referrerpolicy="no-referrer">

Slick.js --- Para los Sliders

<script src="//cdn.jsdelivr.net/npm/slick-carousel@1.8.1/slick/slick.min.js">
<link rel="stylesheet" type="text/css" href="//cdn.jsdelivr.net/npm/slick-carousel@1.8.1/slick/slick.css">


JQuery  --- Necesario para que funciones Slick.JS, por ende se aprovechó para otras funciones. 
<script src="//code.jquery.com/jquery-1.11.0.min.js"></script>
<script src="//code.jquery.com/jquery-migrate-1.2.1.min.js"></script>

Google Fonts para las tipografías. 
@import url('https://fonts.googleapis.com/css2?family=Lato:ital,wght@0,400;1,700&family=Roboto:wght@400;500&display=swap');


Instrucciones. 

Copiar los archivos en una carpeta dentro de un servidor web, ejemplo un servidor de apache. 
Abrir en el navegador la url local asignada. 
Recuerde que debe tener conexión a internet para que se carguen las librerias.
